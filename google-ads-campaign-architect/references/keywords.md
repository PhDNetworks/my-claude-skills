# Keywords Reference

## Match Type Strategy

### Starting Point (New Campaigns)
- **Exact match** [keyword] - Highest intent, start here
- **Phrase match** "keyword" - Moderate reach, good intent
- **Avoid broad match** until you have conversion data

### After 30+ Conversions
- Add broad match with Target CPA/ROAS bidding
- Use broad match modifier equivalents via phrase match
- Monitor search terms report weekly

## Negative Keyword Match Types

### Use Phrase Match Negatives for:
- Broad concepts you never want: "how to"
- Job-related: "jobs", "salary", "course"
- Wrong locations: "london", "manchester"

### Use Exact Match Negatives for:
- Specific irrelevant terms that might match legitimately in other contexts
- Terms where you want precision

### Example Application
```
Campaign-Level Negatives (Phrase Match):
"jobs"
"salary"
"course"
"how to"
"diy"
"london"
"manchester"

Ad Group-Level Negatives (Exact):
[specific irrelevant term for that ad group]
```

## Search Terms Review Process

**Weekly Tasks:**
1. Review Search Terms report (last 7 days)
2. Add converting terms as keywords
3. Add irrelevant terms as negatives
4. Check for new location variations
5. Note any emerging patterns

**Red Flags in Search Terms:**
- High impressions, zero clicks = poor relevance
- Clicks but no conversions = consider negative
- Irrelevant queries matching = tighten match types
