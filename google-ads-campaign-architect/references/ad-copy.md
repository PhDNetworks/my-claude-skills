# Ad Copy Reference

## RSA Structure Requirements

**Responsive Search Ads require:**
- 15 headlines (max 30 characters each)
- 4 descriptions (max 90 characters each)

**Best Practice:**
- Pin location to Headline 1 or 2
- Pin primary CTA to Headline 3
- Include 3+ headlines with keywords
- Include 2+ headlines with USPs
- Mix headline lengths (short + long performs better)

## Compliance Reminders

**Never use:**
- "Best" (unless proven - awards, rankings)
- "Cheapest" (can't substantiate)
- "Guaranteed lowest price" (unless price match policy)
- "100% guaranteed" (vague, unsubstantiated)
- "#1" (unless independently verified)

**Always include:**
- Registration numbers for regulated trades (Gas Safe)
- Accreditation names accurately (NICEIC, not "Niciec")
- Genuine review counts/ratings
- Accurate pricing if mentioned

**UK Spelling:**
- Specialise (not specialize)
- Colour (not color)
- Centre (not center)
- Aluminium (not aluminum)
- Metre (not meter for measurements)

## Character Counts

| Element | Limit | Tip |
|---------|-------|-----|
| Headline | 30 chars | Include 5+ short headlines (under 20) |
| Description | 90 chars | Use full limit for key messages |
| Path 1 | 15 chars | Use for service category |
| Path 2 | 15 chars | Use for location or sub-service |

**Path Examples:**
- /Electrician/Leeds
- /Rewiring/Free-Quote
- /Emergency/24-7
- /Boilers/Installation
- /Roofing/Repairs
