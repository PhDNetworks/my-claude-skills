# My Claude Skills

## Overview
A collection of Claude Skills for digital marketing, SEO, PPC campaigns, and productivity automation. Each skill folder contains specialized prompts and configurations for Claude Code.

## Tech Stack
- Python 100%
- - Claude Code Skills format
 
  - ## Project Structure
  - ```
    /
    ├── Asana-AI-Studio-Expert/     # Asana + AI Studio integration
    ├── content-creator/            # Content creation workflows
    ├── google-ads-campaign-architect/  # Google Ads campaign management
    ├── google-official-seo-guide/  # SEO best practices
    ├── imageforge-pro/             # Image generation workflows
    ├── jlr-roofing-ppc-strategist/ # JLR Roofing PPC campaigns
    ├── keyword-strategy/           # Keyword research and strategy
    ├── local-business-schema-generator/ # Local SEO schema markup
    ├── marketing-demand-acquisition/   # Demand gen campaigns
    ├── notion-expert/              # Notion integration skills
    ├── notion-knowledge-capture/   # Knowledge management
    ├── notion-research-documentation/ # Research documentation
    ├── ossett-dental-ppc-strategist/  # Ossett Dental PPC
    ├── riflebird-cold-email/       # Cold email campaigns
    ├── seo-content-optimizer/      # SEO content optimization
    ├── seo-keyword-cluster-builder/   # Keyword clustering
    ├── technical-seo/              # Technical SEO audits
    └── web-performance-audit/      # Web performance analysis
    ```

    ## Usage
    Each skill folder contains its own configuration. Copy the skill folder to your `.claude/commands/` directory to use with Claude Code.

    ## Key Patterns
    - Each skill is self-contained in its own directory
    - - Skills follow Claude Code's skill format specification
      - - Python-based automation scripts where needed
