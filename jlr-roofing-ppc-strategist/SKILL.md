# JLR Smith Roofing — PPC Strategist

Expert Google Ads management for JLR Smith Roofing & Plastering using a "guerrilla" strategy: high-precision targeting on a £250/month budget.

- **Google Ads Account:** 490-663-7401

## Business Context

**Location:** Leeds, West Yorkshire (+ 10-15 mile radius)

**Services:** Emergency roof repairs, full roof replacements (slate/tile/flat), fascias & soffits, chimney repairs, plastering

**USPs:** Family-run, 10+ years experience, RESTEC-approved, fully insured, 24/7 emergency response, 4.7★ rating, no hidden costs, written guarantees

## Campaign Architecture

| Setting | Value |
|---------|-------|
| Network | Search Only (no Display/Partners) |
| Budget | £250/month (~£8.30/day) |
| Bidding | Maximize Clicks, £6.00 CPC cap |
| Geo | Leeds + 10-15 miles (Presence only) |
| Schedule | 06:00–23:00 daily |

### Ad Groups

**Emergency Repairs (70% budget)**
- Intent: Distress buyers, phone calls, same-day
- Landing: /emergency-roof-repair
- Match types: Exact and Phrase only

**New Roofs (30% budget)**
- Intent: Investment buyers, form submissions
- Landing: /new-roofs
- Match types: Exact and Phrase only

## Target Audience

**Distress Buyers (Emergency)**
- Mobile-first, short urgent queries
- Low price sensitivity, high urgency
- Conversion factor: Speed of response

**Investment Buyers (New Roofs)**
- Desktop/tablet, comparative queries
- Research phase, trust-focused
- Conversion factor: Guarantees, RESTEC, portfolio

## Performance Targets

| Metric | Target |
|--------|--------|
| CTR | >4% |
| Avg CPC | <£6.00 |
| CPL | £60–£125 |
| Leads/month | 2–4 verified |
| Quality Score | 7+ |

**ROI Context:** Repair job = £200–£500. New roof = £5,000+. One new roof lead covers annual ad spend.

## Keyword Strategy

Use **Exact Match** and **Phrase Match** ONLY. Never use Broad Match.

**Emergency Priority Keywords:**
```
[emergency roof repair leeds], [24 hour roofer leeds], [urgent roofer leeds]
[roof leak repair leeds], [storm damage roof repair], [emergency roofer near me]
```

**New Roofs Priority Keywords:**
```
[new roof installation leeds], [roof replacement leeds], [reroofing quote leeds]
[slate roof replacement], [flat roof installation leeds]
```

**Negative Keywords:** See [references/negative-keywords.md](references/negative-keywords.md) for full list including competitor brands, aggregators, and wrong intent terms.

## Ad Copy Guidelines

**Emergency Ads** — Tone: Urgent, reassuring
- Headlines: 24/7 Emergency, Roof Leak? Call Now, Fast Response, On-Site Within Hours
- Themes: Speed, availability, fully insured, local experts

**New Roof Ads** — Tone: Professional, trustworthy
- Headlines: 20-Year Guarantee, Free Quotes, RESTEC Approved, Quality Replacements
- Themes: Guarantees, family-run trust, no hidden costs

**Extensions:** Sitelinks (Emergency, New Roofs, Reviews, Contact), Callouts (24/7, Free Quotes, Insured, RESTEC), Call Extension (critical for mobile)

## Weekly Optimisation (Fridays)

1. Search Terms Report → add negatives, flag competitors
2. Keyword Performance → pause high-spend zero-conversion terms
3. Ad Performance → review RSA assets, test new headlines if CTR <3%
4. Budget Pacing → adjust if over/under
5. Device/Schedule → review mobile vs desktop, adjust bids

## Monthly Cycle (10th)

1. Export full performance report
2. Calculate CPL and lead quality
3. Review conversion tracking health
4. Prepare client summary

## Quarterly Review (Mar/Jun/Sep/Dec 10th)

Full keyword audit, competitor analysis, landing page audit, budget review, strategy adjustment.

## Critical Rules

**Never:**
- Use Broad Match keywords
- Enable Display Network or Search Partners
- Run ads 24/7 (no overnight response)
- Launch Performance Max until budget >£750/month and conversions >30/month
- Ignore Search Terms Report for >7 days

**Escalate if:**
- Spend exceeds £10/day for 3+ consecutive days
- CTR drops below 2%
- CPC exceeds £8 consistently
- Conversion tracking shows errors

## Conversion Tracking

- Phone calls >30 seconds from call extension
- Form submissions via thank-you page (/thank-you)
- All leads must be verified (answered, qualified)
